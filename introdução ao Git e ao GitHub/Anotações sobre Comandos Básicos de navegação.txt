Comandos Básicos de Navegação (Windows)

- Mudar nome de pasta -> cd 
- Listar as pastas -> Dir
- Criar pastas/arquivos -> mkdir
- Deletar pastas/arquivos -> del/rmdir

Comandos Básicos de Navegação (Unix)

- Mudar nome de pasta -> cd 
- Listar as pastas -> Ls
- Criar pastas/arquivos -> mkdir
- Deletar pastas/arquivos -> rm/rf

Comando para mover -> mv
